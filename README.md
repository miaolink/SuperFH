# SuperFH
烽火光猫Telnet管理工具 - 功能强大的光猫配置助手
✨ 核心功能：
• 🔑 一键获取超级密码 - 自动开启Telnet并获取管理员账号密码
• 🌍 地区切换 - 支持POST和Telnet两种方式切换地区，批量处理多台设备
• 🔧 配置修改 - TR069 WAN修复、RMS服务器配置修改
• 🔄 恢复出厂设置 - 支持POST方式恢复出厂
• 🔓 工程账号管理 - 一键开启工程账号
• 📡 地区检测 - 快速检测当前设备地区和运营商
• 🤖 批量处理 - 全自动批量切换地区，支持多台设备


💡 特色亮点：
• 支持移动/联通/电信三大运营商
• 智能MAC地址识别和自动重试
• 2K/4K屏幕自适应显示方案
• 简洁易用的图形界面
• 详细的运行日志输出


📦 适用场景：
- 光猫地区切换和配置管理
- 批量设备维护
- 网络调试和故障排查
